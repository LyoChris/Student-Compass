Pe main se afla folderul pentru deploy pentru frontend, care a fost facut pe vercell (care permite deploy doar de pe branchul principal).

Pe branchul deploy se afla deploy-ul de backend, care este hostat pe render.

Pe branchul ai se afla folderul de ai care este hostat local folosing ngrok (serviciu care ne ofera un link unic prin care putem accesa deploy-ul de ai de oriunde).

Avem automat partea de build si deploy pentru frontend si backend, aceste servicii fiind oferite de vercell, respectiv render.
